## 第一阶段：注册小程序账号

### 1.1 访问注册页面

**网址**: https://mp.weixin.qq.com/

**步骤**:
1. 点击右上角"立即注册"
2. 选择账号类型：**小程序**
3. 进入注册流程

### 1.2 填写账号信息

**必填信息**:
```
邮箱：未注册过微信开放平台的邮箱
      推荐：QQ邮箱、163邮箱、Gmail
密码：8-20位，包含大小写字母、数字、符号
确认密码：再次输入
验证码：邮箱收到的6位数字
```

**注意事项**:
- ✅ 邮箱必须是未注册过的
- ✅ 密码强度要足够
- ✅ 验证码有效期10分钟

### 1.3 选择主体类型

#### 方案A：个人开发者（推荐，快速上线）

**优点**:
- ✅ 注册快速（1-3天审核）
- ✅ 无需营业执照
- ✅ 免费

**缺点**:
- ❌ 无法开通微信支付
- ❌ 部分类目受限
- ❌ 无法认证

**适用场景**:
- 快速验证产品
- 初期免费模式
- 无支付需求

**注册信息**:
```
主体类型：个人
主体名称：你的真实姓名
身份证号：18位身份证号码
管理员微信：扫码绑定你的微信号
管理员手机：你的手机号
验证方式：短信验证码
```

#### 方案B：企业开发者（全功能）

**优点**:
- ✅ 支持微信支付
- ✅ 全部功能可用
- ✅ 可认证，信誉更好
- ✅ 支持更多类目

**缺点**:
- ⏳ 审核时间长（3-7天）
- 💰 需要营业执照
- 💰 认证费用300元/年

**适用场景**:
- 需要支付功能
- 正式商业运营
- 品牌推广

**注册信息**:
```
主体类型：企业
主体名称：公司全称
营业执照号：统一社会信用代码
法定代表人：姓名、身份证号
对公账户：银行账号信息
管理员信息：同个人开发者
```

### 1.4 填写小程序信息

**必填信息**:
```
小程序名称：疯神榜
           （如被占用，可选：疯神榜学习、疯神榜AI）
小程序简介：社交+AI辅助学习平台，找到同水平的学习伙伴
           （4-120字）
服务类目：教育 → 在线教育 → 学习辅导
头像：上传小程序图标（建议使用产品Logo）
```

**名称规则**:
- 4-30个字符
- 必须唯一（不能与已有小程序重名）
- 每年可修改2次

**类目选择**:
```
一级类目：教育
二级类目：在线教育
三级类目：学习辅导
```

### 1.5 等待审核

**审核时间**:
- 个人开发者：1-3个工作日
- 企业开发者：3-7个工作日

**审核结果通知**:
- 微信消息通知
- 短信通知
- 邮件通知

**审核通过后**:
- 获得**小程序AppID**
- 可以开始开发
