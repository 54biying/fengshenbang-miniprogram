# 微信小程序注册 - 快速参考

## 🚀 5分钟快速开始

### 第1步：注册账号（5分钟）
1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册" → 选择"小程序"
3. 填写邮箱和密码
4. 选择"个人开发者"
5. 填写个人信息
6. 等待审核（1-3天）

### 第2步：获取工具（10分钟）
1. 下载微信开发者工具
2. 扫码登录
3. 创建新项目

### 第3步：导入项目（5分钟）
1. 项目目录：`/workspace/projects/fengshenbang-miniprogram`
2. AppID：注册后获得（或使用测试号）
3. 勾选"不校验合法域名"

### 第4步：连接后端（5分钟）
1. 确认后端服务运行中（端口 9000）
2. 修改 `app.js` 中的 baseUrl：
   ```javascript
   baseUrl: 'http://localhost:9000/api'
   ```
3. 测试登录功能

### 第5步：测试发布（30分钟）
1. 测试所有功能
2. 上传代码
3. 提交审核
4. 等待上线

## 📝 关键配置位置

### 小程序 baseUrl
- 文件：`app.js`
- 代码：
  ```javascript
  globalData: {
    baseUrl: 'http://localhost:9000/api'
  }
  ```

### 后端 API 端点
- 基础URL：`http://localhost:9000/api`
- 主要接口：
  - 登录：`/api/auth/login`
  - 注册：`/api/auth/register`
  - 测试：`/api/test/questions`
  - 匹配：`/api/matching/recommend`
  - 拜师礼：`/api/mentorship/gifts`

## ⚠️ 重要提示

### 开发模式 vs 生产模式

| 特性 | 开发模式 | 生产模式 |
|------|---------|---------|
| 域名校验 | 关闭 | 开启 |
| 协议 | HTTP | HTTPS |
| baseUrl | localhost | 云服务器 |
| 环境 | 本地 | 外网 |

### 个人开发者限制
- ❌ 无法开通微信支付
- ❌ 部分高级功能受限
- ✅ 可以正常上线
- ✅ 免费使用

### 企业开发者优势
- ✅ 支持微信支付
- ✅ 全部功能可用
- ✅ 品牌信誉更好
- ⚠️ 需要营业执照

## 💡 建议

### 最小化验证方案（无需支付）
1. 使用个人开发者账号注册
2. 使用模拟支付（当前已实现）
3. 先上线核心功能（AI测试 + 匹配）
4. 收集用户反馈
5. 后续升级为企业开发者开通支付

### 成本最低方案
- 使用免费云服务器（阿里云/腾讯云新用户）
- 使用免费域名（如 .tk、.ml）或无需域名（内网穿透）
- 使用免费 SSL 证书（Let's Encrypt）
- 总成本：¥0（第一年）

### 最快上线方案
- 使用个人开发者账号（审核快）
- 不配置服务器域名（开发模式）
- 使用内网穿透（ngrok）
- 总时间：3-5天
