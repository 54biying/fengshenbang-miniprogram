## 1. 注册小程序账号

### 1.1 访问注册页面
- 网址：https://mp.weixin.qq.com/
- 点击"立即注册" → 选择"小程序"

### 1.2 填写账号信息
```
邮箱：未被注册过的邮箱（推荐使用QQ邮箱）
密码：大写字母+小写字母+数字+符号，8位以上
确认密码：再次输入密码
验证码：输入邮箱收到的验证码
```

### 1.3 选择主体类型
| 类型 | 要求 | 权限 | 时长 |
|------|------|------|------|
| **个人开发者** | 身份证 | 无微信支付 | 1-3天 |
| **企业开发者** | 营业执照 | 全部功能 | 3-7天 |

**推荐**：先注册个人开发者，快速上线，后续可以升级为企业。

### 1.4 完善主体信息
#### 个人开发者：
- 主体名称：真实姓名
- 身份证信息：如实填写
- 管理员微信号：你的微信号
- 管理员手机号：你的手机号

#### 企业开发者：
- 主体名称：公司名称
- 营业执照号：统一社会信用代码
- 法定代表人信息
- 对公账户信息

### 1.5 小程序信息
- **小程序名称**：疯神榜（需要唯一，可能需要调整）
  - 建议名称：疯神榜、疯神榜学习、疯神榜AI
- **小程序简介**：社交+AI辅助学习平台，找到同水平的学习伙伴
- **服务类目**：教育 → 在线教育 → 学习辅导

### 1.6 等待审核
- 个人：1-3个工作日
- 企业：3-7个工作日

## 3. 配置开发环境

### 3.1 导入项目
1. 点击"+"号创建项目
2. 填写项目信息：
   - **项目名称**：疯神榜
   - **目录**：选择 `/workspace/projects/fengshenbang-miniprogram`
   - **AppID**：注册后获得（选择"测试号"可以暂时不填）
   - **开发模式**：小程序
   - **后端服务**：不使用云服务

### 3.2 配置项目详情
点击右上角"详情" → "本地设置"：
- ✅ 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
- ✅ 启用 ES6 转 ES5
- ✅ 增强编译
- ✅ 使用 npm 模块

### 3.3 安装依赖
```bash
cd /workspace/projects/fengshenbang-miniprogram
npm install
```

### 3.4 构建项目
1. 在开发者工具中点击"工具" → "构建 npm"
2. 等待构建完成

## 4. 连接后端服务

### 4.1 检查后端服务状态
```bash
# 检查端口 9000 是否被占用
lsof -i :9000

# 查看 FastAPI 服务状态
ps aux | grep uvicorn

# 测试 API 接口
curl http://localhost:9000/api/health
```

### 4.2 修改小程序 baseUrl

#### 方法1：直接修改 app.js
```javascript
// 编辑 /workspace/projects/fengshenbang-miniprogram/app.js
globalData: {
  baseUrl: 'http://localhost:9000/api'  // 开发环境
  // baseUrl: 'https://101.34.64.175/api'  // 生产环境
}
```

#### 方法2：使用环境变量（推荐）
```javascript
// 创建 config.js
const config = {
  baseUrl: process.env.API_BASE_URL || 'http://localhost:9000/api'
}

module.exports = config

// 在 app.js 中引用
const config = require('./config.js')
globalData: {
  baseUrl: config.baseUrl
}
```

### 4.3 测试连接
1. 修改配置后保存
2. 在开发者工具中刷新
3. 测试登录功能：
   - 点击"登录"按钮
   - 输入手机号
   - 获取验证码
   - 完成登录

### 4.4 外网访问配置

#### 方案1：使用云服务器（推荐）
1. 购买云服务器（阿里云、腾讯云、华为云）
2. 部署后端服务到云服务器
3. 配置 HTTPS（使用 Let's Encrypt）
4. 在小程序管理后台配置服务器域名

#### 方案2：使用内网穿透（开发测试）
```bash
# 安装 ngrok
# 下载地址：https://ngrok.com/download

# 启动 ngrok
ngrok http 9000

# 获得 HTTPS 地址（例如：https://abc123.ngrok.io）
# 在小程序中配置：
baseUrl: 'https://abc123.ngrok.io/api'
```

## 5. 申请微信支付

### 5.1 个人开发者限制
- ❌ 个人开发者**无法**开通微信支付
- ⚠️ 如果需要支付功能，必须升级为企业开发者

### 5.2 企业开发者流程

#### 步骤1：开通微信支付
1. 登录小程序管理后台
2. 进入"微信支付"页面
3. 点击"开通微信支付"
4. 填写商户信息：
   - 商户名称：公司名称
   - 营业执照：上传营业执照
   - 结算银行账户：对公账户

#### 步骤2：配置支付参数
获得以下信息：
- **商户号**：mch_xxxxxxxxx
- **API密钥**：自行设置（32位字符串）
- **APIv3密钥**：自行设置（32位字符串）
- **商户证书**：下载证书文件

#### 步骤3：配置后端支付
```javascript
// 在后端配置微信支付参数
const wechatPayConfig = {
  appId: '你的小程序AppID',
  mchId: '商户号',
  apiKey: 'API密钥',
  apiV3Key: 'APIv3密钥',
  certPath: '/path/to/cert.p12'
}
```

#### 步骤4：修改小程序支付代码
```javascript
// 调用微信支付
wx.requestPayment({
  timeStamp: '',
  nonceStr: '',
  package: '',
  signType: 'MD5',
  paySign: '',
  success(res) {
    console.log('支付成功', res)
  },
  fail(err) {
    console.error('支付失败', err)
  }
})
```

### 5.3 临时解决方案
如果暂时无法开通微信支付，可以：
1. **使用模拟支付**：当前代码已实现
2. **接入第三方支付**：
   - 支付宝（个人可接入）
   - Stripe（国际支付）
3. **先上线免费版本**：不包含支付功能

## 6. 提交审核发布

### 6.1 代码上传
1. 在开发者工具中点击"上传"
2. 填写版本号和项目备注
3. 等待上传完成

### 6.2 提交审核
1. 登录小程序管理后台
2. 进入"版本管理" → "开发版本"
3. 点击"提交审核"
4. 填写审核信息：
   - 测试账号：提供测试账号
   - 功能页面截图：上传截图
   - 补充说明：描述小程序功能

### 6.3 等待审核
- 个人开发者：1-3个工作日
- 企业开发者：3-7个工作日

### 6.4 发布上线
审核通过后：
1. 进入"版本管理" → "审核版本"
2. 点击"发布"
3. 小程序正式上线

## 附录：常见问题

### Q1: 小程序名称被占用怎么办？
A: 添加修饰词，如：
- 疯神榜学习
- 疯神榜AI
- 疯神榜助手

### Q2: 开发工具无法连接后端？
A: 检查以下几点：
1. 后端服务是否启动
2. baseUrl 配置是否正确
3. 是否关闭了域名校验
4. 防火墙是否阻止连接

### Q3: 个人开发者如何开通支付？
A: 无法开通，解决方案：
1. 升级为企业开发者
2. 使用第三方支付
3. 先上线免费版本

### Q4: 如何修改小程序名称？
A:
1. 登录管理后台
2. 设置 → 基本信息
3. 点击"修改"
4. 每年只能修改2次

### Q5: 服务器域名配置失败？
A: 检查：
1. 域名是否已备案
2. 是否使用 HTTPS
3. SSL 证书是否有效
4. 域名是否解析到服务器
